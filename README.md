# Bayesian Inference – ME712

This repository contains my exercises, projects, and practical work for the course **ME712 – Bayesian Inference** at **Unicamp**.

The material is based on the book:  
**“Bayes’ Rules” – Alicia A. Johnson, Miles Q. Ott, Mine Dogucu**.

## Contents

- **Exercises** – Solved exercises from the course and the textbook.
- **Projects** – Practical projects applying Bayesian inference concepts.
- **Data Analyses** – Examples of data analysis using Bayesian methods.

## Purpose

The goal of this repository is to:

- Organize and document my learning throughout the undergraduate course.
- Serve as a reference for studying Bayesian inference.
- Showcase practical applications of Bayesian concepts.

## Tools & Technologies

- Python / R (depending on the exercise/project)
- Jupyter Notebooks / R Markdown
- Libraries: `bayesrules`, `tidyverse`, `janitor`, `kableextra`, `knitr`, `ggplot2` (R)
