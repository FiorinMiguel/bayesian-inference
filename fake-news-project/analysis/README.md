# Detecting Fake News with Bayesian Inference

## Description

This project uses **Bayesian inference** to estimate the probability that a news article is fake based on the presence of an exclamation mark in its title.

---

## Objective

Compute:

P(Fake | Exclamation)

---

## Data

- Source: `bayesrules` package  
- 150 articles labeled as `fake` or `real`  

---

## Methodology

- Define the prior  
- Compute the likelihood  
- Apply Bayes' Theorem  
- Obtain the posterior  

---

## Tools

R, tidyverse, ggplot2, janitor, kableExtra
